# Intent-Based Chatbot (v1)

This is a simple rule-based chatbot with intent classification.

## Features

* Intent classification using keyword matching
* Supports:

  * Greeting
  * Time queries
  * Weather queries
  * Add tasks
  * View tasks
* Simple task storage (in-memory)

## How to Run

```bash
python chatbot.py
```

Type `exit` to quit.

## Example Inputs

* "hi"
* "what's the time"
* "add study DSA"
* "show my tasks"
